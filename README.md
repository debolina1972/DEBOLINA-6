# Sales Forecasting System

## 📌 Project Overview
The Sales Forecasting System predicts future sales using Machine Learning and Time Series Analysis techniques.

The system helps businesses improve inventory management and business planning.

---

## 🎯 Objectives
- Predict future sales
- Identify seasonal trends
- Improve inventory planning
- Support business decision-making

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- Matplotlib
- Seaborn

---

## 📂 Dataset Features
- Date
- Product ID
- Store ID
- Units Sold
- Revenue
- Promotion
- Holiday Indicator

---

## ⚙️ Algorithms Used
- Linear Regression
- ARIMA
- Moving Average
- Random Forest Regressor
- LSTM

---

## 🚀 Project Workflow
1. Data Collection
2. Data Preprocessing
3. Time Series Analysis
4. Feature Engineering
5. Model Training
6. Forecasting
7. Evaluation

---

## 📊 Evaluation Metrics
- MAE

- RMSE
